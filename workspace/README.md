# LawMind `workspace/`（仓库内示例盘）

本目录用于 **本地开发与演示**：引擎与桌面会把「工作区根」指向此处或指向 Electron 用户数据下的目录。

- **应提交**：模板、`playbooks/`、`cases/` 中的示例案件、`MEMORY.md` / `LAWYER_PROFILE.md` / `FIRM_PROFILE.md` 等**非敏感样例**。
- **勿提交**：任务、草稿、会话、审计、渲染产物等——见本目录 `.gitignore`。

完整仓库结构说明见 **[LAWMIND-REPO-LAYOUT.md](../docs/LAWMIND-REPO-LAYOUT.md)**。
