## 完成项

### 1. UI 收敛式重设计（v0.2）

- 替换初版"功能堆砌"风格，转向克制的深色暖铜专业风格

- 排版：PingFang SC 中文优先，正文 14px/1.7 行高

- 自定义 Markdown 渲染（标题/列表/粗体/行内代码/分隔线，零外部依赖）
- 列表密度优化：去卡片化，透明底 + hover 左侧指示线
- 空态引导精简：纯文字场景卡，"有什么可以帮您？"

### 2. 统一设置面板（v0.3）

- 侧边栏极简化：品牌栏 + 齿轮按钮 + 助手 select + 项目药丸 + 折叠工作记录

### 4. 工作记录折叠与按助手过滤

## 技术备忘

- 构建验证：`node_modules/.bin/vite build` 通过，`pnpm exec oxlint` 零错误零警告
- CSS token 化：所有颜色/圆角/阴影在 `:root` 定义，便于后续主题切换
- 零运行时依赖新增（设计约束：不引入新 npm 包到渲染进程）
- Markdown 渲染为纯字符串替换 + DOM 直出，不依赖 marked/remark 等库
